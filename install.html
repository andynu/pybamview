get the source code

Note: PyBamView requires Python2.7.

To install pybamview, first obtain the source code by either:
  • Downloading the tarball {% highlight python %} tar -xzvf mgymrek-pybamview-[version].tar.gz cd mgymrek-pybamview-[version] {% endhighlight %}
  • Downloading the zip file {% highlight python %} unzip mgymrek-pybamview-[version].zip cd mgymrek-pybamview-[version] {% endhighlight %}
  • Cloning the git repository {% highlight python %} git clone https://github.com/mgymrek/pybamview.git cd pybamview {% endhighlight %}

install requirements

Then, make sure you have the required python packages installed:
  • Flask
  • pandas
  • pyfasta
  • pysam
You can install these using easy_install: {% highlight python %} sudo easy_install Flask pandas pyfasta pysam {% endhighlight %}

install

At the top of the PyBamView source directory, type: {% highlight python %} sudo python setup.py install {% endhighlight %} To make sure it was installed, type: {% highlight python %} pybamview --help {% endhighlight %} and ensure that the help message comes up. Then move on to the usage page to get started!

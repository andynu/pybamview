Pybamview

Browser-based application for viewing bam alignments

Note: PyBamView requires Python2.7.

easy installation

The easiest way to install PyBamView is using pip: {% highlight bash %} pip install pybamview {% endhighlight %} This will install the latest PyBamView version listed in PyPi, and any missing libraries it depends on.

To use the "Export to PDF" feature, you will also need to install rsvg-convert: {% highlight bash %} sudo apt-get install librsvg2-bin {% endhighlight %}

installing from source

obtain the source code

To install PyBamView, first obtain the source code by either:
  • Downloading the tarball {% highlight bash %} tar -xzvf mgymrek-pybamview-[version].tar.gz cd mgymrek-pybamview-[version] {% endhighlight %}
  • Downloading the zip file {% highlight bash %} unzip mgymrek-pybamview-[version].zip cd mgymrek-pybamview-[version] {% endhighlight %}
  • Cloning the git repository {% highlight bash %} git clone https://github.com/mgymrek/pybamview.git cd pybamview {% endhighlight %}

install requirements

Then, make sure you have the required python packages installed:
  • Flask
  • pandas
  • pyfasta
  • pysam
You can install these using easy_install: {% highlight bash %} sudo easy_install Flask pandas pyfasta pysam {% endhighlight %} If you want to use the "Download PDF" feature, you must have rsvg-convert installed (see above).

install

At the top of the PyBamView source directory, type: {% highlight bash %} sudo python setup.py install {% endhighlight %} To make sure it was installed, type: {% highlight bash %} pybamview --help {% endhighlight %} and ensure that the help message comes up. Then move on to the usage page to get started!
